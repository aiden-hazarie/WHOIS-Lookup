## WHOIS Lookup
Description: A program that looks up information about a website (like its owner, registration date, and hosting provider). Similar to checking a license plate to find a car’s owner, this tool reveals public details about domain names.

## Key Features
- Retrieves registration records for domains (e.g., who owns example.com).
- Helps verify website legitimacy or investigate suspicious domains.
- Uses public databases to provide transparency.
